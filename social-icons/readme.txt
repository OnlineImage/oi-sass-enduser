Use of the social web icons is based on this, however do not use the .fc-webicon class just use .webicon
http://zurb.com/playground/social-webicons
Example:
            <a href="# target="_blank" class="webicon facebook large">Like us on Facebook</a>
            <a href="#" target="_blank" class="webicon twitter">Follow us on Twitter</a>
            <a href="#" target="_blank" class="webicon googleplus small">GooglePlus</a>
Using class small or large will change the size of the webicons, using a class of medium calls a different web icon so don't use that
